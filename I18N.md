## How It Works

1. **Language Switcher** - Dropdown in top-right corner
2. **Auto-Save** - Selected language saved to localStorage
3. **Data Attributes** - Elements with `data-i18n="key"` are auto-translated

## Adding a New Language

### 1. Add translations to `i18n.js`:

```javascript

  // ...existing languages

  it: {  // Italian
    'main.title': 'PartKit',
    'main.subtitle': 'Strumenti PDF per musicisti di brass band',
    'tool.splitter.name': 'Divisore PDF',
```

## Translation Keys

### Main Page
- `main.title` - Site title
- `main.subtitle` - Site subtitle
- `tool.*.name` - Tool names
- `tool.*.desc` - Tool descriptions

### Common
- `common.back` - Back button
- `common.download` - Download button
- `common.processing` - Processing message
- `common.error` - Error label
- `common.cancel` - Cancel button

### Splitter
- `splitter.upload.*` - Upload area
- `splitter.detected.title` - Detected splits title
- `splitter.filename.*` - Filename input
- `splitter.download.*` - Download buttons
- `splitter.merge.*` - Merge buttons
- `splitter.split.pages` - Split pages button
- `splitter.delete` - Delete button

### Combiner
- `combiner.upload.*` - Upload area
- `combiner.first.alone` - First page alone toggle
- `combiner.crop.*` - Crop controls
- `combiner.zip.filename` - ZIP filename

### Merger
- `merger.step1.*` - Step 1 section
- `merger.step2.*` - Step 2 section
- `merger.preview.title` - Preview title
- `merger.download.*` - Download buttons

### Assembler
- `assembler.upload.*` - Upload area

- `assembler.filename.*` - Filename input
- `assembler.assemble` - Assemble button

## Using Translations in HTML

Add `data-i18n="key"` attribute to any element:

```html
<h1 data-i18n="main.title">PartKit</h1>
<p data-i18n="main.subtitle">PDF tools for brass band musicians</p>
<button data-i18n="common.download">Download</button>
```

For placeholders:
```html
<input placeholder="Search..." data-i18n="common.search">
```

## Using Translations in JavaScript

```javascript
import { t, setLanguage, getCurrentLanguage } from './i18n.js';

// Get translation
const title = t('main.title');

// Change language
setLanguage('de');

// Get current language
const lang = getCurrentLanguage();  // 'en', 'de', etc.
```

## Contributing Translations

Want to add a language? Please contribute!

1. Copy the English (`en`) object in `i18n.js`
2. Translate all values (keep keys unchanged)
3. Add your language to `getAvailableLanguages()`
4. Submit a pull request!

## Language Priority

1. User's selected language (from dropdown)
2. Previously saved language (localStorage)
3. English (fallback)
